WSL2设置静态IP

 浮叶蓝空     2021年09月23日     2207  

WSL2每次重启后都会被重新分配IP，每次重启电脑后想要访问WSL里面的服务还需要重新查一下IP，不能忍！
网上搜到一个方案，记录一下https://zhuanlan.zhihu.com/p/357050093

windows下创建一个批处理（.bat）文件来完成操作

给WSL绑定一个IP为192.168.2.100，给宿主机绑定一个IP为192.168.2.200

  1. @echo off
  2. setlocal enabledelayedexpansion
  4. ::先停掉可能在跑的wsl实例
  5. wsl --shutdown ubuntu
  7. if !errorlevel! equ 0 (
  8.     ::检查WSL有没有我需要的IP
  9.     wsl -u root ip addr | findstr "192.168.2.100" > nul
  10.     if !errorlevel! equ 0 (
  11.         echo wsl ip has set
  12.     ) else (
  13.         ::IP不存在则绑定IP
  14.         wsl -u root ip addr add 192.168.2.100/24 broadcast 192.168.2.255 dev eth0 label eth0:1
  15.         echo set wsl ip success: 192.168.2.100
  16.     )
  18.     ::检查宿主机有没有我需要的IP
  19.     ipconfig | findstr "192.168.2.200" > nul
  20.     if !errorlevel! equ 0 (
  21.         echo windows ip has set
  22.     ) else (
  23.         ::IP不存在则绑定IP
  24.         netsh interface ip add address "vEthernet (WSL)" 192.168.2.200 255.255.255.0
  25.         echo set windows ip success: 192.168.2.200
  26.     )
  27. )
  28. pause

电脑开机后用管理员身份跑一次这个脚本就好了
