本地搭建Spring Initializr服务器(运行在docker中)

环境

win10 + VMware15.0.2 + CentOS7 + docker18.09.7

说明：

本次要将Spring Initializr运行在docker容器中，我电脑是将docker安装在了虚拟机CentOS系统中，如果自己电脑中已有docker环境直接运行步骤2即可。

步骤

  1. 安装docker[可安装在本机，也可安装在虚拟机中]

  2. 启动start_spring_io 容器(映射本机端口，我映射到了本机8888端口。8080常用，防止冲突)

        docker run -tid --name spring-initializr -p 8888:8080 pollyduan/start_spring_io:latest

  3. 测试

  • 浏览器访问

  • idea中新建springboot项目

疑问

start.spring.io如果能构建成功，可以直接在本地以springboot项目启动运行，简单方便。但我构建时执行到yarn build 总会报错。？？？，还没有找到解决办法。
