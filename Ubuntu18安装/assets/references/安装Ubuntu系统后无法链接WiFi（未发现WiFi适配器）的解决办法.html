安装Ubuntu系统后无法链接WiFi（未发现WiFi适配器）的解决办法

标签： Ubuntu  ubuntu

       1、首先得保证新装的系统可以联网是吧，但是在没有网线的情况下，我也就只能出此下策了。可以使用联网的手机和USB线来连接到电脑上，开启USB共享网络，就可以上网了。

       2、更改软件源。将下图中的“下载自”改成合适的地址，可以选阿里、腾讯啥的。

       3、进入“附加驱动”选项卡，如果网络正常，且源设置正常，那么一般能搜到对应的无线网卡驱动，安装之后再重启电脑就可以了。要注意你的bois是否位于UEFI且开启了secure boot，必须把secure boot关掉，因为专用驱动是闭源的，而ubuntu开源，微软认为“闭源系统是不安全的，所以闭源的软件在开启secure boot的时候不能启用”。

       如果此时还是不能显示WiFi列表的话，可以尝试在终端输入如下的命令：

  1. sudo apt-get update  
  2. sudo apt-get install bcmwl-kernel-source

----------------------------------------------------手打分割线----------------------------------------------------

       以上就是我解决问题的粗略步骤，不过这个问题也有可能是别的原因造成的，由于是第一次装Ubuntu的系统，所以比较小白，记录这个问题以供自己以后参考~
